Gesture Control for Instagram Reels

A computer vision project that allows controlling Instagram Reels using hand gestures.

Features

* Next Reel (Scroll Down)
* Previous Reel (Scroll Up)
* Like 👍
* Pause / Play ⏯️
* Volume Control 🔊
* Mute 🔇
* Exit Gesture 

Tech Used

* Python
* OpenCV
* CVZone
* PyAutoGUI
* PyQt5

▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run GUI:
   python your_gui_file.py

3. Run gesture system:
   python gesture_control.py
